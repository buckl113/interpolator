Name Search

path: addrBook/search

sample uri: http://namesearch.com/addrBook/search?fname=joe&lname=jackson&maxRec=389

  • maxRec= (maxrec) 99
  • fname= (db/person/fname) First name of person

  • lname= (db/person/lname#tech_desc) Last Name of person stored in addressbook.table.person.lname in primary oracle database. Must Match legal name as shown on drivers license or passport.

    { 'person' : 
{'lname' : 'myname' } 
}
    • Type= (db/person/lname#type) string
    • *len= (db/person/lname#len) 50
      • From Database desc
